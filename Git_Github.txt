Comandos basicos

git ini  = Nos permite indicar que se va a crear un proyecto nuevo
git add <file> = permite pasar los archivos de working directory a staging area
git status = permite ver en que estado estàn los archivos
git commit = permite pasar los archivos de staging area al repositorio
git push= Sirve para subir a un escritorio remoto, ejemplo subir los archivos a un servidor lo cual otros desarrolladores que tienen permiso podrán verlo
git pull = este comando trae los cambios que han hecho otros desarrolladores
git clone = permite crear una copia desde donde está el proyecto a la computadora


Ejemplo   
            ---      working directory  (lugar donde se trabaja con todos los archivos
             git add
            ---       staging area   ---  (lugar donde se van agregando todos los archivos o preparando todos los archivos para el guardado)
                                      git commit
                       repository    --- (Cuando estàn listo para ser guardados los archivos, estos son almacenados en el repositorio)
